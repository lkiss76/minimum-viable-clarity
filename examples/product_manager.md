# MVC Example — Product Manager

## Intent
Translate user feedback into actionable feature insights.

## Context
The PM is preparing a quarterly roadmap and needs structure from raw feedback logs.

## Criteria
- Group findings into themes  
- Highlight high-frequency pain points  
- Provide 2–3 suggested feature directions  
- Avoid implementation details  

## Exemplar
- Theme: Onboarding confusion  
- Pain point: Users struggle with first login  
- Feature direction: Guided onboarding flow
